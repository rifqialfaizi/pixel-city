Application for download a picture from flickr by arean in a mapKit
## Getting Started

``` bash
git clone https://github.com/rifqialfaizi/pixel-city.git
cd pixel-city
open pixel-city.workspace # or xed .

- Run (⌘ + R) or
- Test (⌘ + U)
```
